# 简单的bilibili弹幕发送者反查工具

## 使用方法
0. **拉取**此项目
1. 准备好**Python**环境（**requests**和**bs4**库自行安装）
2. 填好派森文件内的参数（**CID**和**弹幕关键字**）
3. 运行派森文件，得到目标**CRC32校验码**
4. 使用浏览器打开HTML，输入刚刚获取的**CRC32校验码**，单击**计算**即可

## 注意：由算法可知，结果**不保证**100%正确
